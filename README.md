# obapps4

An updated graphical editor for the Openbox window manager's per-application settings (`rc.xml`).

## Setup

See [DEVELOPMENT](DEVELOPMENT.md) for setup and installation.

## Usage

```bash
obapps
```

By default obapps edits `~/.config/openbox/rc.xml` (or whichever config file Openbox was
started with). Pass a path to edit a different file:

```bash
obapps ~/.config/openbox/myrc.xml
```
